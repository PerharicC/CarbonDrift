Simulation Notes

  1. The updated fragmentation option for GZ has not been substantialy tested so far (especially in plotting), thus use it with caution.
  2. The updated GridRun object for running massive simulations has not been substantialy tested so far, thus use it with caution. However, for simulations without fragmentation, most computers should have no problem running the simualtions in normal mode. Simulations on 1 degree grid cells (number of particles of order ≳ 10^4) have not shown any memory problems, even with added horizontal advection.
